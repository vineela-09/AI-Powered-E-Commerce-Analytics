# 🛒 AI-Powered E-Commerce Analytics (Python | SQL | Power BI)

## 📌 Project Overview

AI-Powered E-Commerce Analytics is an end-to-end data analytics project designed to analyze online retail data and generate business insights using Python, SQL, and Power BI.

The project focuses on cleaning and analyzing sales data, understanding customer purchasing behavior, evaluating product performance, and building interactive dashboards to support data-driven business decisions.

---

## 🎯 Objectives

- Analyze sales performance and revenue trends.
- Identify top-selling products and categories.
- Understand customer purchasing behavior.
- Track customer retention and repeat purchases.
- Monitor key business KPIs.
- Build interactive dashboards for business reporting.
